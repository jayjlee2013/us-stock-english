# US Stock 영어 학습 프로젝트

## 목적
미국주식 투자에 필요한 영어 용어·문장을 초급→중급→고급 단계로 학습하는 웹앱.
참고 컨셉: 책 "미국주식 필수사이트 읽는 영어" — 실제 증권사 앱, 재무제표, 뉴스, 실적발표에서 쓰이는 영어 위주.

## 구조
- `index.html` — 단일 파일 학습 앱 (용어 데이터 + UI + 퀴즈 모두 포함)
  - 데이터: `TERMS` 배열 (en, ko, desc, ex, exKo, lv, cat, src?)
  - 레벨: 1=초급, 2=중급, 3=고급
  - 카테고리: 기초/거래/재무·실적/시장·경제/차트·분석/세금
  - 모드: 학습(카드 목록) / 오늘의 5단어(일일 자동 추천, 날짜 시드 결정론적) / 플래시카드 / 퀴즈 / 실전 문장
  - 발음 듣기: Web Speech API(`speechSynthesis`) 사용, 외부 라이브러리 없음. 학습/오늘의5단어/플래시카드/실전문장 전체에 🔊 버튼
  - 진도(외운 단어)는 `usstock_known`, 오늘의 5단어 선택 결과는 `usstock_daily`, 최고 달성 등급은 `usstock_stage` — 모두 localStorage
- **성장 시스템**: 각 레벨 70%(`STAGE_GOAL`) 암기 시 승급 — 🌱새싹→📈주린이 탈출→💼실전 투자자→🏆월가 마스터
  - 등급이 오르면 "집중 레벨"이 초급→중급→고급으로 이동
  - 오늘의 5단어: 집중 레벨 4개 + 타 레벨 1개 자동 선별
  - 퀴즈(전체 필터일 때): 집중 레벨 미암기 7개 + 외운 단어 복습 3개. 복습 오답 시 '외웠어요' 자동 해제

## 규칙
- 외부 라이브러리 없이 순수 HTML/CSS/JS 단일 파일 유지
- 용어 추가 시 반드시 예문(ex)과 한글 번역(exKo) 포함
- **출처 원칙 (사용자 요청)**: 실제 사이트에서 현재 쓰이는 단어만 추가. 실제 화면에서 확인한 용어는 `src` 필드에 출처 기재
  - 확인된 출처 (2026-07-16): Yahoo Finance 종목/통계/뉴스 화면, Finviz 펀더멘털 표, stockanalysis.com
  - 실전 문장에는 수집 당일 실제 헤드라인 포함 가능 (출처+날짜 표기)
